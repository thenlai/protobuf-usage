# Grammar of Protobuf

Protobuf v2 or Protobuf v3

# Protobuf V3

For details, please visit [official guide](https://developers.google.com/protocol-buffers/docs/proto3).

## types

| type    | default value |
|---------|---------------|
| string  | empty string  |
| byte    | empty byte    |
| bool    | false         |
| numerics| zero, 0       |
| enum    | first defined, must be 0 |
| message |   not set     |
